Affordance annoatation tool

To use this tool, deploy it on a server and update all three instances of the password in sqlconnect.php, insert.php and app.php to your own MySQL server password. 
You can change the main data used in the review by loading the SQL dump file, deleting the available data and replacing it with your own. 
The specifics of the database are found in the thesis document titled "Supporting affordance detection and extraction from online product reviews"
